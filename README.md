# AWS Certified Solutions Architect – Associate (SAA-C03)

A study resource repository for the **AWS Certified Solutions Architect – Associate (SAA-C03)** certification exam.

## About

This repository contains reference material for preparing for the AWS SAA-C03 exam. The certification validates the ability to design and implement distributed systems on AWS.
